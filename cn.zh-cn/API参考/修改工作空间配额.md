## 请求示例

修改工作空间配额

```
PUT  https://{endpoint}/v1/{project_id}/workspaces/{workspace_id}/quotas

{
  "quotas" : [ {
    "workspace_id" : "***9cd9ea8a5432cbcd6496e57839***",
    "resource" : "exemlProject.gpu_duration",
    "quota" : 10
  } ]
}
```
