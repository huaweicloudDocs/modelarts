## 请求示例

同步数据集状态获取

```
GET https://{endpoint}/v2/{project_id}/datasets/{dataset_id}/sync-data/status
```

**状态码： 200**

OK

```

  "status" : "COMPLETED",
  "dataset_id" : "gfghHSokody6AJigS5A"
}
```
