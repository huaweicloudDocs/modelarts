# 删除授权<a name="DeleteAuthorizations"></a>

## 功能介绍

删除指定用户的授权或者删除全量用户的授权。

## URI

DELETE /v2/\{project\_id\}/authorizations

```
DELETE https://{endpoint}/v2/{project_id}/authorizations?user_id=****d80fb058844ae8b82aa66d9fe****
```
