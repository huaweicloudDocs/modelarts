# 创建ModelArts委托<a name="CreateModelArtsAgency"></a>

## 功能介绍

创建包含OBS、SWR、IEF等依赖服务的ModelArts委托。

## 响应参数

无

## 请求示例

创建ModelArts委托

```
POST https://{endpoint}/v2/{project_id}/agency

{
  "agency_name_suffix" : "iam-user01"
}
```

## 响应示例

**状态码： 200**

OK

```
{
  "agency_name" : "ma_agency_iam-user01"
}
```
