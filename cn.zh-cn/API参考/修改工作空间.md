## 请求示例

修改工作空间

```
PUT  https://{endpoint}/v1/{project_id}/workspaces/{workspace_id}

{
  "name" : "my_workspace",
  "description" : "It is my workspace",
  "auth_type" : "INTERNAL",
  "grants" : [ {
    "user_name" : "my_iam_user"
  } ]
}
```
