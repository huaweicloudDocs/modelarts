# 查询处理任务详情<a name="DescribeProcessorTask"></a>

## 功能介绍

查询处理任务详情，支持查询“特征分析”任务和“数据处理”两大类任务。可通过指定路径参数“task\_id”来查询某个具体任务的详情。

## URI

GET /v2/\{project\_id\}/processor-tasks/\{task\_id\}

