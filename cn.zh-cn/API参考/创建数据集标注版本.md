## 请求示例

创建数据集标注版本

```
{
  "version_name" : "V004",
  "version_format" : "Default",
  "description" : "",
  "clear_hard_property" : true
}
```

## 响应示例

**状态码： 201**

Created

```
{
  "version_id" : "sntOdOuB0D9C6fC4TXs"
}
```
