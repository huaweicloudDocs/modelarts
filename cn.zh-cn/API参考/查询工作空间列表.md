# 查询工作空间列表<a name="ListWorkspaces"></a>

## 功能介绍

查询工作空间列表，响应消息体中包含详细信息。

## 请求示例

查询工作空间列表

```
GET https://{endpoint}/v1/{project_id}/workspaces?limit=2&offset=3&sort_by=name&order=desc&enterprise_project_id=0
```
