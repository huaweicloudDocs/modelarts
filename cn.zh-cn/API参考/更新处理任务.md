# 更新处理任务<a name="UpdateProcessorTask"></a>

## 功能介绍

更新处理任务，支持更新“特征分析”任务和“数据处理”两大类任务，仅支持更新任务的描述。可通过指定路径参数“task\_id”来更新某个具体任务。

## URI

PUT /v2/\{project\_id\}/processor-tasks/\{task\_id\}
