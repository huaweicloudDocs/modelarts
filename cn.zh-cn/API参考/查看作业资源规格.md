## 示例<a name="section136871714386"></a>

如下以查看训练作业的资源规格为例。

-   请求示例

    ```
    GET    https://endpoint/v1/{project_id}/job/resource-specs?job_type=train
    ```


-   成功响应示例

    ```
    {
        "is_success": true,
        "spec_total_count": 3,
        "specs": [
            {
                "spec_id": 1,
                "core": 8,
                "cpu": 8,
                "gpu_num": 2,
                "gpu_type": "P100",
                "spec_code": "dls.vm.gpu.p100",
                "unit_num": 1,
                "max_num": 2,
                "storage": "4TB",
                "interface_type": 1,
                "no_resource": true
            }
        ]
    }
    ```
